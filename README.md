# Nexus | สถาปัตยกรรมแห่งการประมวลผล

เว็บไซต์แหล่งข้อมูลการศึกษาเชิงลึกด้านสถาปัตยกรรมคอมพิวเตอร์ ระดับพรีเมียม (Single-Page Application) ที่ครอบคลุมเนื้อหาตั้งแต่ ประวัติศาสตร์คอมพิวเตอร์, ส่วนประกอบฮาร์ดแวร์, พื้นฐานการเขียนโค้ด, การตั้งค่าสภาพแวดล้อม (OS Setup), ระบบโฮสติ้ง และ การปรับแต่ง SEO

## 🚀 การทดสอบรันบนเครื่องของคุณ (Local Development)

เนื่องจากโปรเจกต์นี้เป็นหน้าเว็บไซต์แบบคงที่ (Static Website) ที่ใช้เพียง HTML, CSS (Vanilla) และ JavaScript พื้นฐาน คุณสามารถเปิดใช้งานได้ทันทีโดย:
1. ดับเบิลคลิกที่ไฟล์ `index.html` เพื่อเปิดในเว็บเบราว์เซอร์ของคุณ
2. หรือใช้เครื่องมือเช่น **Live Server** (ส่วนเสริมยอดนิยมใน VS Code) เพื่อรันเว็บเซิร์ฟเวอร์จำลองเพื่อประสบการณ์การพัฒนาที่ราบรื่น

---

## 🌐 วิธีการนำเว็บไซต์ขึ้นโฮสติ้งฟรี (Free Hosting Deployment)

คุณสามารถนำโปรเจกต์นี้ขึ้นไปออนไลน์บนอินเทอร์เน็ตให้ทุกคนสามารถเข้าถึงได้ **ฟรีตลอดชีพ** ผ่านแพลตฟอร์มที่รองรับ Static Website ยอดนิยม 3 วิธี ดังนี้:

### วิธีที่ 1: การโฮสต์ผ่าน GitHub Pages (แนะนำที่สุด)
เป็นบริการฟรีจาก GitHub ที่มั่นคง ปลอดภัย และเหมาะสำหรับโปรเจกต์ HTML/CSS พื้นฐาน

**ขั้นตอนการทำ:**
1. สมัครสมาชิกและเข้าสู่ระบบที่ [GitHub](https://github.com/)
2. สร้าง Repository ใหม่โดยคลิก **"New"** (ตั้งชื่ออะไรก็ได้ เช่น `nexus-website`) และให้ตั้งค่าเป็น **Public** (สาธารณะ)
3. อัปโหลดไฟล์และโฟลเดอร์ทั้งหมดในโปรเจกต์นี้ (`index.html`, `style.css`, `script.js` และโฟลเดอร์ `assets`) ลงไปใน Repository ที่เพิ่งสร้าง
4. ในหน้า Repository ของคุณ ให้คลิกที่เมนู **Settings** (รูปฟันเฟือง) ด้านบน
5. ที่เมนูด้านซ้าย ให้มองหาและคลิกที่ **Pages**
6. ภายใต้หัวข้อ **Build and deployment** ให้ดูที่ **Source** แล้วเลือกเป็น `Deploy from a branch`
7. ในหัวข้อ **Branch** ให้กดเลือก `main` (หรือ `master`) และช่องถัดมาเป็น `/ (root)` จากนั้นกดปุ่ม **Save**
8. รอประมาณ 1-2 นาที คุณจะเห็นข้อความสีเขียวพร้อมลิงก์เว็บไซต์ของคุณ (รูปแบบ: `https://[ชื่อบัญชีคุณ].github.io/[ชื่อ-repo]`)

---

### วิธีที่ 2: การโฮสต์ผ่าน Netlify (วิธีที่ง่ายที่สุด - แค่ลากแล้ววาง)
เหมาะสำหรับผู้ที่ไม่ต้องการใช้ระบบ Git เพียงแค่มีโฟลเดอร์งานก็ขึ้นเว็บได้ทันที

**ขั้นตอนการทำ:**
1. เข้าไปที่ [Netlify](https://www.netlify.com/) และทำการสมัครสมาชิก (Sign Up)
2. ไปที่หน้า Dashboard ของคุณ เลือกเมนู **Sites** ที่แถบด้านซ้าย
3. เลื่อนลงมาด้านล่างสุด คุณจะพบกล่องที่เขียนว่า **"Drag and drop your site output folder here"**
4. ให้คุณลาก **"โฟลเดอร์โปรเจกต์ทั้งหมด"** (โฟลเดอร์ที่ครอบไฟล์ index.html อยู่) มาปล่อยลงในกล่องนี้
5. ระบบจะทำการอัปโหลดและ Deploy ให้โดยอัตโนมัติภายในไม่กี่วินาที
6. คุณจะได้ลิงก์เว็บไซต์ทันที (สามารถเข้าไปแก้ชื่อโดเมนได้ที่เมนู Domain Management -> Options -> Edit site name)

---

### วิธีที่ 3: การโฮสต์ผ่าน Vercel (อัปเดตไว ทันสมัย)
แพลตฟอร์มที่ได้รับความนิยมสูงมากในหมู่นักพัฒนายุคใหม่ ทำงานได้รวดเร็วปานสายฟ้า

**ขั้นตอนการทำ:**
1. สมัครสมาชิกที่ [Vercel](https://vercel.com/) (แนะนำให้ล็อกอินผ่านบัญชี GitHub)
2. เมื่อเข้าสู่หน้า Dashboard ให้คลิกปุ่ม **"Add New..."** มุมขวาบน และเลือก **"Project"**
3. หากคุณนำโค้ดไปฝากไว้บน GitHub แล้ว คุณสามารถกดปุ่ม **Import** โปรเจกต์นั้นได้เลย
4. ให้ข้ามการตั้งค่า Framework (ปล่อยให้ระบบเลือกเป็น Other) แล้วกดปุ่ม **Deploy**
5. รอไม่ถึง 1 นาที เว็บไซต์ของคุณจะออนไลน์และใช้งานได้ทันที พร้อมโดเมนรูปแบบ `https://[ชื่อโปรเจกต์].vercel.app`

## 🎨 โครงสร้างเทคโนโลยีที่ใช้ (Tech Stack)
* **HTML5**: โครงสร้างเอกสารเชิงความหมาย (Semantic Structure)
* **CSS3**: การจัดรูปแบบหน้าจอ ดีไซน์ Dark Mode ระดับพรีเมียม จัดวางด้วย Flexbox/Grid และใช้งาน CSS Variables
* **JavaScript**: รองรับลูกเล่นแอนิเมชัน Intersection Observer (Scroll Reveal) และ Navigation Effects
* **Typography**: ดึงระบบฟอนต์จาก Google Fonts (`Prompt` สำหรับภาษาไทยให้อ่านง่าย และ `Inter` / `Playfair Display` สำหรับความหรูหรา)
